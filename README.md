# Проект
## Загрузка проекта
Для загрузки проекта нажмите зеленую кнопку "Code" и выбирите ZIP. Архив распакуйте в любую директорию.
## Установка библиотек
Установка библиотек и зависимостей перейдите в терминале в папку с проектом и выполните команду

    python -m pip install -r requirements.txt

Примечание: фреймворк для нейронных сетей весит около 400 мб!
## Запуск проекта
Для запуска проекта откройте файл "main.py" в любом редакторе кода и запустите его.
По умолчанию вместе с python устанавливается редактор кода IDLE.
Если у вас нет других редакторов кода, в строке поиска Windows наберите "IDLE".
В меню выберите "Open" и откройте файл "main.py". Запусп произовдится с помощью  F5.
